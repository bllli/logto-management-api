# Verifications
(*Verifications*)

## Overview

Endpoints for creating and validating verification records, which can be used in Profile routes.

### Available Operations

* [CreateByPassword](#createbypassword) - Create a record by password
* [CreateByVerificationCode](#createbyverificationcode) - Create a record by verification code
* [VerifyByCode](#verifybycode) - Verify verification code
* [CreateBySocial](#createbysocial) - Create a social verification record
* [VerifyBySocial](#verifybysocial) - Verify a social verification record
